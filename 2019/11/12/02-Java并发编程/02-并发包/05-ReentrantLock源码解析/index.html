内部类分析

  ReentrantLock内部总共有三个内部类,并且FairSync和NonfairSync都是Sync的具体实现

Sync

  Sync是抽象同步器,它继承了AQS,实现了部分非公平和公平的方法,后续的FairSync和NonfairSync都继承Sync并实现了lock和tryAcquire方法
